在桌面鼠标右键打开 Git Bash Here，执行 git config --global user.name "sunhong" 命令设置用户名，执行 git config --global user.email "845036815@qq.com" 命令设置邮箱。使用了 --global 选项，该命令只需要运行一次，即可永久生效。设置的用户名和邮箱地址会被写入到 C:\Users\SJJ\.gitconfig 文件中，这个文件是 Git 的全局配置文件，配置一次即可永久生效，使用记事本打开此文件，查看自己曾经对 Git 做了哪些全局性的配置。首先在项目的根目录中通过鼠标右键打开 Git Bash Here，执行 git init 命令将当前的目录转化为 Git 仓库，git nit 命令会创建一个名为 .git 的隐藏目录，这个 .git 目录就是当前项目的 Git 仓库。然后执行 git add . 命令将项目里的所有新增和修改过的文件加入暂存区中。最后执行 git commit -m "init project" 命令将暂存区的文件提交到 Git 仓库，其中 "" 双引号中的内容描述这次提交都做了哪些事情。
